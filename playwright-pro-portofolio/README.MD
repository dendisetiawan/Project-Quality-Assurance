# Playwright Pro Portfolio - Prof

Repository ini berisi rangkaian pengujian otomatis (Automation Testing) menggunakan **Playwright** dan **TypeScript**. Project ini dibuat untuk mendemonstrasikan kemampuan dalam menangani berbagai skenario pengujian web modern.

## 🚀 Fitur Pengujian
Saat ini, suite pengujian mencakup:
* **Authentication Flow**: Pengujian skenario login secara end-to-end.
* **E-commerce Flow**: Simulasi proses checkout barang dari awal hingga akhir.
* **Dynamic Elements Handling**: Menangani elemen web yang muncul secara dinamis menggunakan *Waiting Strategy* (Logic: `waitForSelector` & `hidden state`).
* **Page Object Model (POM)**: Implementasi struktur kode yang modular dan mudah dipelihara.

## 🛠️ Tech Stack
* **Framework**: Playwright
* **Language**: TypeScript
* **Structure**: Page Object Model (POM)
* **Tools**: VS Code, Git/GitHub
